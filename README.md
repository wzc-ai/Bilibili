================================================================================
                            B站数据工具箱
                         Bilibili Data Toolkit
================================================================================

项目简介
--------
B站数据工具箱是一个基于Flask的B站（Bilibili）数据抓取与分析平台，提供UP主
视频数据爬取、弹幕数据采集与词频分析等功能。

功能特性
--------
1. UP主视频爬取
   - 获取指定UP主的视频列表
   - 支持分页查询
   - 自动处理WBI签名验证

2. 弹幕数据采集
   - 爬取视频弹幕内容
   - 支持弹幕数据导出（CSV格式）

3. 弹幕分析
   - 词频统计分析
   - 词语可视化分析

4. Web界面
   - 简洁美观的可视化界面
   - 实时数据展示
   - 响应式设计

项目结构
--------
Bilibili/
├── app.py                    # Flask主程序
├── templates/
│   └── index.html            # Web界面模板
├── up主视频爬取/
│   ├── spider.py             # 视频爬虫脚本
│   └── w_rid.js             # WBI签名JS代码
├── 弹幕爬取与分析/
│   ├── 弹幕.py               # 弹幕爬虫
│   ├── 词语.py               # 词频分析
│   └── 弹幕.csv              # 弹幕数据
└── 弹幕1.csv                 # 示例弹幕数据

环境依赖
--------
- Python 3.8+
- Flask
- requests
- execjs
- PyExecJS (需要Node.js运行时)

安装步骤
--------
1. 安装Python依赖:
   pip install flask requests pyexecjs

2. 安装Node.js (用于执行WBI签名JS代码)

3. 启动服务:
   python app.py

4. 访问Web界面:
   打开浏览器访问 http://127.0.0.1:5000

使用方法
--------
1. 在Web界面输入UP主ID或BV号
2. 选择需要的功能（视频爬取/弹幕分析）
3. 查看分析结果

注意事项
--------
- 本工具仅供学习研究使用，请勿滥用
- 请遵守B站的服务条款和使用协议
- 爬取数据时请合理控制频率，避免对服务器造成压力

技术说明
--------
- 使用B站公开API获取数据
- WBI签名通过JavaScript代码执行
- 弹幕数据以CSV格式存储便于分析
